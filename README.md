#  Biblioteca Virtual

Aplicación de consola desarrollada en C# para gestionar libros de manera sencilla mediante programación orientada a objetos y buenas prácticas de arquitectura de software.

##  Funcionalidades

- Registrar libros
- Buscar libros
- Listar libros
- Prestar libros
- Devolver libros

## 🛠️ Tecnologías utilizadas

- C#
- .NET
- Consola

##  Arquitectura

El proyecto está organizado por responsabilidades:

- UI (interfaz de consola)
- Servicios (lógica del sistema)
- Repositorios (manejo de datos)

Aplicando principios SOLID y separación por capas.

Este proyecto utilizó herramientas de inteligencia artificial únicamente como apoyo académico para resolver dudas, mejorar documentación y reforzar conceptos de programación y arquitectura de software.
